## 1. IBSA 공식 규칙 요약

### 개인전
- **11점** 선취, **2점차** 필수 (듀스 방식)
- **3세트 중 2세트 선승** (Best of 3)
- 준결승/결승: 5세트 중 3세트 선승 (Best of 5) - 추후 구현
- 골 = 2점, 상대 폴트 = 1점
- 타임아웃: 세트당 1회, 60초
- 서브: 2회 연속 후 교대

### 팀전 (IBSA 공식)
- **31점** 선취, **2점차** 필수
- **1세트** 단판
- 팀 3명이 로테이션 (각 3회 서브 후 교대)
- 교체: 경기 중 1회 가능
- 하나의 테이블에서 팀 vs 팀으로 진행

### 기타
- 모든 선수 불투명 고글 착용 (등급 무관 동일 조건)
- 심판: 주심 + 부심
- 관중 조용히 (공 소리 청취 필요)

## 2. 토론 합의 사항

### 2.1 개인전 규칙 (합의)
- **11점, 2세트 선승, 최대 3세트** → 고정 (선택 UI 제거)
- 2점차 필수
- 타임아웃: 세트당 1회, 60초
- 기존 평터/반칙 기록 유지

### 2.2 팀전 규칙 (합의)
- IBSA 방식 채택: **31점, 1세트, 팀 vs 팀 단일 경기**
- 기존 NxN 개별 경기 방식 → IBSA 로테이션 방식으로 변경
- 점수 기록은 팀 단위 (개인전과 동일한 UI, 31점으로 변경)
- 팀 구성: 3명

### 2.3 랜덤 팀리그전 (합의)
- 팀 간 경기도 IBSA 팀전 규칙 적용 (31점, 1세트)
- 기존 NxN 개별 경기 방식 제거 → 단일 31점 경기로 통일

### 2.4 선수 관리 (합의)
- 대회별 선수 등록 기능 추가
- 전역 선수 풀도 유지 (기존 선수 가져오기 가능)
- 대회 상세 > 참가자 탭에서:
  - "전역 선수에서 추가" (기존 방식)
  - "새 선수 직접 등록" (대회 내 로컬 등록)

### 2.5 대회 생성 UI 변경
- 승점/세트 선택 UI 제거 (규칙 고정)
- 대회 유형 선택 시 자동으로 규칙 설정:
  - 개인전 → 11점, 2세트 선승
  - 팀전 → 31점, 1세트
  - 랜덤 팀리그전 → 31점, 1세트
