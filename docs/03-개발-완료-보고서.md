## 2. 구현된 기능

### 2.1 공통 기반 (shared/)
- **타입 시스템**: Tournament 중심 데이터 모델 (33개 타입/인터페이스)
- **Firebase 훅**: 14개 커스텀 훅 (대회, 경기, 팀, 스케줄, 알림, 즐겨찾기 등)
- **유틸리티**: 점수 판정, 순위 산출, PIN 해시/검증, 디바이스 ID
- **공통 컴포넌트**: 연결 상태 표시, 로딩 스피너
- **인증**: PIN 기반 관리자/심판 인증 (SHA-256 해시)

### 2.2 관리자 모드 (admin/)
- **대시보드**: 대회 목록, 생성, 삭제
- **대회 생성**: 유형(개인전/팀전/랜덤팀리그), 게임 설정
- **대회 상세 (5개 탭)**:
  - 참가자: 선수 선택, 랜덤 팀 생성 (Fisher-Yates)
  - 대진표: 라운드로빈 자동 생성, 심판/경기장 배정
  - 스케줄: 시간/경기장 자동 배정
  - 경기 현황: 필터, 상태 관리, 대회 시작/일시정지/완료
  - 순위: 개인전/팀전 순위 자동 산출
- **선수 관리**: CRUD
- **심판 관리**: CRUD + PIN 설정
- **경기장 관리**: CRUD + 심판 배정

### 2.3 심판 모드 (referee/)
- **로그인**: 대회 선택 → 심판 선택 → PIN 입력 (3단계)
- **배정 경기 목록**: 상태별 그룹핑, 시간순 정렬
- **개인전 점수 기록**: 전체화면, 큰 버튼, 세트 관리, 평터/반칙/타임아웃
- **팀전 점수 기록**: 개별 경기 점수 인라인 입력, 자동 팀 승자 판정

### 2.4 관람 모드 (spectator/)
- **대회 목록**: 진행중/완료 필터
- **대회 관람 (4개 탭)**:
  - 실시간: 진행중 경기 실시간 점수, 즐겨찾기 별표
  - 대진표: 풀리그 그리드, 팀전 매치업 목록
  - 순위: 테이블 (스크린리더 최적화)
  - 히스토리: 완료 경기 기록
- **경기 상세 관람**: 읽기 전용 실시간 점수판
- **즐겨찾기**: 선수 관리, 활성 경기 표시
